# Card War Game

# How the game works
Deck is shuffle and shared between the total number of players.
For every Round, each player places a card face-up, winner is determine
by the card with the higher rank.
If there's a tie, all tie players play another round and winner takes
everything in the pot
User is prompted to enter number of rounds and number of players.
User is prompted to Enter the Q key to quit at each round or any key 
to continue the game.

# Description
	1.	Game supports 2 - 4 players only
	2.	Round Play is automatic
	3.	10,000 Round Cap

# How to buid the game
	1.	Create a class of card and create an enum with name of rank and suit
	2.	Create the deck class and shuffle it.
	3.	Create the hand class to hold card queue for each player
	4.	Create a player class
	5.	Put logic together in game engine
